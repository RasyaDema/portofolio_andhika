## ‚ö†Ô∏è TROUBLESHOOTING

### Icon tidak muncul?

1. Periksa nama file (harus huruf kecil)
2. Periksa lokasi file (harus di `assets/images/`)
3. Restart aplikasi Flutter
4. Run `flutter clean` lalu `flutter pub get`

### Icon terlalu besar/kecil?

Icon otomatis di-resize ke 140x140px di aplikasi.

### Icon tidak transparan?

Pastikan download PNG dengan background transparan atau pilih icon lain.

---

## üí° TIPS

1. **Gunakan icon dengan style yang sama** (flat design lebih baik)
2. **Pilih icon berwarna** agar terlihat di background hitam
3. **Download ukuran besar** (512px atau lebih) untuk kualitas terbaik
4. **Compress image** jika file size terlalu besar (gunakan TinyPNG.com)
